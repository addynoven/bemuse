# v3 — Social & Competitive

> **Goal:** Players compete, share, and build a community around the game.

## Exit Criteria
- [ ] User accounts with persistent profiles
- [ ] Global and friend leaderboards per song
- [ ] Shareable replay links
- [ ] Rate and comment on community charts
- [ ] Basic multiplayer (2P local or online)
