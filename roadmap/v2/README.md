# v2 — Content & Discovery

> **Goal:** Users can find, discover, and play any song without knowing a YouTube URL.

## Exit Criteria
- [ ] Search any song by title/artist
- [ ] Browse trending/popular charts
- [ ] See what others are playing
- [ ] Multiple chart algorithms (not just lyrics)
- [ ] Word-level granularity for dense charts

## Files

| File | Topic |
|------|-------|
| [song-search.md](./song-search.md) | Search YouTube via SDK |
| [trending-charts.md](./trending-charts.md) | Popular songs, weekly top 10 |
| [chart-algorithms.md](./chart-algorithms.md) | Lyrics, rhythm, melody, hybrid generators |
| [word-level-notes.md](./word-level-notes.md) | Notes for every syllable/word |
| [preview-system.md](./preview-system.md) | 15s audio preview before playing |
| [history-page.md](./history-page.md) | Full play history with stats |
| [favorites.md](./favorites.md) | Like/bookmark songs |
| [offline-cache.md](./offline-cache.md) | Cache audio + charts for offline play |
