# MusicStream Rhythm — Roadmap

> A rhythm game powered by musicstream-sdk, forked from Bemuse.
> Auto-generates beatmania-style charts from synced lyrics and audio streams.

---

## Version Philosophy

| Version | Theme | Focus |
|---------|-------|-------|
| **v0.x** | *Proof of Life* | Does it work at all? |
| **v1.x** | *Playable* | Can a stranger enjoy a full song? |
| **v2.x** | *Discovery* | Can users find and play any song? |
| **v3.x** | *Community* | Can players share, compete, and create? |
| **v4.x** | *Power User* | Can advanced users customize everything? |
| **v5.x** | *Platform* | Is this a self-sustaining ecosystem? |

---

## Current Status

**Latest:** v0.x (in development)
- Bemuse fork builds and runs
- SDK proxy fetches audio + lyrics
- Auto-generated bmson charts from lyric timing
- Single hardcoded demo song (Queen — We Will Rock You)
- Per-note seek-and-play audio (hacky but functional)
